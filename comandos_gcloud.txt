SET PROJECT:

    gcloud config set project ageless-command-181413

PARA HACER DEPLOY:

    gcloud app deploy app_smalledata_v2.yaml --version=1

    A veces es necesario indeicar el proyecto:

    gcloud app deploy app_smalledata_v2.yaml --version=1 --project="ageless-command-181413"

PARA OBTENER LA URL DE LA WEB:

    gcloud app browse -s smalledata-v1
    
    que da como resultado:
    https://smalledata-v1-dot-ageless-command-181413.uc.r.appspot.com

PARA VER LOS LOGS DE LA CONSOLA:

    gcloud app logs tail -s default

SET PROJECT:

    gcloud config set project apuestas-seguras

PARA HACER DEPLOY:

    gcloud app deploy app.yaml --version=1

    A veces es necesario indeicar el proyecto:

    gcloud app deploy app.yaml --version=1 --project="apuestas-seguras"

PARA OBTENER LA URL DE LA WEB:

    gcloud app browse -s apuestas-seguras
    
    que da como resultado:
    https://apuestas-seguras.appspot.com

PARA VER LOS LOGS DE LA CONSOLA:

    gcloud app logs tail -s default
